This is the class website for Core 2 Iteraction Studio at Parsons The New School, Spring 2023 with Marie Otsuka.

The site structure and materials are influenced by the work of Laurel Schwulst, Minkyoung Kim, Chris Hamamoto, and previous classes I have taught.

The site is a static site generated by 11ty and the full code for this site is available on Github.

The website will evolve as we progress through our class — please check back often for up to date resources.
