Modular Letter

As we dive back into coding, let’s get familiar with drawing with CSS.

  1. In a 3 by 5 grid in Figma, sketch a modular letter like this
  2. Recreate your letter in CSS using CSS Grid
  3. Add an animation to your letter
  4. Make multiple letters to create a 5-letter word

2. Create and style your grid letter

Use the techniques you learned with CSS Grid to setup your structure.
Provide appropriate class names so that you can style your grid blocks accordingly.
Have fun styling your blocks with other CSS techniques, including CSS transforms, border-radius, and others.

  • CSS Transforms Lecture Slides
  • CSS Border Radius Lecture Slide
  • CSS Gradients
  • CSS Filter Effects and the Drop Shadow tool, (Use CSS Text shadow for Text)
